# Express / Car Management

A simple car management system with CRUD (create, read, update, delete) functionality

## Ayub Abdullah

- [@Ayub Abdullah](https://www.github.com/ayubabdullah)

## Tech Stack

**client:**

- ejs
- bootstarp
- css
  **server:**
- express
- mongoDB
- mongoose

## Run Locally

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
